# ZenIME

**一款极简、基于物理粒子特效的 Windows 输入法状态悬浮窗。**

## 功能特性

* ** 粒子物理引擎：** 基于 HTML5 Canvas 打造。文字由流动的粒子汇聚而成，鼠标划过时会像烟雾般自然消散，随后柔和地重组。
* ** 全局无焦点干扰：** 采用 `uiohook-napi` 实现底层键盘监听，精准捕捉 `Shift` 键切换。**绝不抢占当前窗口焦点**，无论是沉浸式写代码还是全屏打游戏，都互不干扰。
* ** HUD 沉浸式设置：**
    * **鼠标右键**点击悬浮窗，即可呼出隐藏的设置面板。
    * **自定义本命字：** 随心设定显示的汉字（如“火”、“禅”、“雷”）。
    * **智能拼音转换：** 内置 `pinyin-pro`，英文模式下自动将汉字转换为拼音（如 'Huo'）。
    * **主题色定制：** 支持 RGB 全色域拾取。
* ** 幽灵模式：** 窗口背景全透明，始终置顶（Always on Top），且鼠标点击可穿透（仅右键响应设置），仿佛悬浮在屏幕上的全息投影。

## 🛠 技术栈

* **Electron:** 应用运行时框架
* **HTML5 Canvas:** 视觉特效与物理算法
* **uiohook-napi:** 系统级全局键盘钩子 (Global Hook)
* **pinyin-pro:** 轻量级汉字转拼音工具

## 安装与开发指南

### 前置要求
* Node.js (推荐 LTS 版本)
* npm

### 1. 安装依赖
```bash
npm install
```
### 2.启动方式
```bash
npm start
```
### 3.打包为exe文件
```bash
npm run dist
```
目标文件出现在 `dist` 目录下, 文件名 `ZenIME-1.0.0.exe`。
